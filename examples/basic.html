Basic HTML to PDF Conversion

This example demonstrates the basic HTML to PDF conversion capabilities of the html2pdf converter. It shows how standard HTML and CSS are rendered into a high-quality PDF document with proper typography, layout, and pagination.

Quick Start: Run cargo run --example complete_example to see the programmatic API in action, or use the CLI: html2pdf basic.html -o basic.pdf

Features Demonstrated

  • Typography: Custom fonts, sizes, and line heights
  • Colors: Text and background colors
  • Layout: Margins, padding, and borders
  • Tables: Structured data with borders and backgrounds
  • Code formatting: Monospace fonts and syntax highlighting
  • Page breaks: Explicit page breaks for sections

Code Example

Here's how to use the library programmatically:

use html2pdf::{html_to_pdf, Config, Input};

fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Create HTML input
    let html = r#"<h1>Hello World</h1>"#;
    let input = Input::Html(html.to_string());
    
    // Configure output
    let config = Config::default();
    
    // Convert to PDF
    let pdf_bytes = html_to_pdf(&input, &config)?;
    std::fs::write("output.pdf", pdf_bytes)?;
    
    Ok(())
}
